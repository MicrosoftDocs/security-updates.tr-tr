Security Bulletin

Microsoft Güvenlik Bülteni MS11-049 - Önemli
============================================

Microsoft XML Düzenleyicisi'ndeki Güvenlik Açığı Bilginin Açığa Çıkmasına Neden Olabilir (2543893)
--------------------------------------------------------------------------------------------------

Yayım Tarihi: 14 Haziran 2011 Salı | Güncelleştirme Tarihi: 15 Şubat 2012 Çarşamba

**Sürüm:** 2.4

### Genel Bilgiler

#### Yürütmeyle İlgili Özet

Bu güvenlik güncelleştirmesi Microsoft XML Düzenleyicisi'ndeki özel olarak bildirilen bir güvenlik açığını giderir. Bu güvenlik açığı, bir kullanıcı özel hazırlanmış bir Web Hizmeti Bulma (.disco) dosyasını bu bültende listelenen etkilenen yazılımlardan biriyle açarsa bilginin açığa çıkmasına neden olabilir. Bu güvenlik açığının, bir saldırganın doğrudan kod yürütmesine veya kullanıcı haklarını yükseltmesine izin vermeyeceğini unutmayın; ancak, etkilenen sistemden daha fazla yararlanmak amacıyla kullanılacak bilgileri üretmek için kullanılabilir.

Bu güvenlik güncelleştirmesi Microsoft InfoPath 2007 ve Microsoft InfoPath 2010'un tüm desteklenen sürümleri; SQL Server 2005, SQL Server 2008 ve SQL Server 2008 R2'nin tüm desteklenen sürümleri; Microsoft Visual Studio 2005, Microsoft Visual Studio 2008 ve Microsoft Visual Studio 2010'un tüm desteklenen sürümleri için Önemli olarak derecelendirilmiştir. Daha fazla bilgi için, bu bölümdeki **Etkilenen ve Etkilenmeyen Yazılımlar** alt bölümüne bakın.

Güvenlik güncelleştirmesi, XML Düzenleyicisi'nin bir Web Hizmeti Bulma (.disco) dosyasındaki dış varlıkları çözümleme biçimini düzelterek bu güvenlik açığını giderir. Bu güvenlik açığı hakkında daha fazla bilgi için, bir sonraki bölüm olan **Güvenlik Açığı Bilgileri** altında, ilgili güvenlik açığı girdisinin Sık Sorulan Sorular (SSS) alt bölümüne bakın.

**Öneri.** Microsoft, müşterilerin güncelleştirmeyi ilk fırsatta uygulamalarını önerir.

**Bilinen Sorunlar.**[Microsoft Bilgi Bankası makalesi 2543893](http://support.microsoft.com/kb/2543893), bu güvenlik güncelleştirmesini yükleyen müşterilerin karşılaşabileceği bilinen sorunları belgelemektedir. Bu makale ayrıca, bu sorunlar için önerilen çözümleri de içermektedir.

#### Etkilenen ve Etkilenmeyen Yazılımlar

Aşağıdaki yazılımlar, hangi sürümlerinin etkilendiğini belirlemek amacıyla sınanmıştır. Diğer sürümler destek ömrünü tamamlamıştır veya bu açıktan etkilenmemektedir. Yazılımınızın sürümünün destek ömrünü belirlemek için [Microsoft Destek Ömrü](http://go.microsoft.com/fwlink/?linkid=21742) Web sitesini ziyaret edin.

XML Dış Varlıklarını Çözümleme Güvenlik Açığı - CVE-2011-1280
-------------------------------------------------------------

<span></span>
Microsoft XML Düzenleyicisi'nin özel hazırlanmış XML dosyalarını işleme biçiminde bilginin açığa çıkması güvenlik açığı bulunmaktadır.

Bu güvenlik açığını Ortak Güvenlik Açıkları ve Etkilenmeler listesinde standart girdi olarak görüntülemek için, bkz: [CVE-2011-1280](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2011-1280).

#### XML Dış Varlıklarını Çözümleme Güvenlik Açığı - CVE-2011-1280 için Azaltıcı Etkenler

Azaltıcı etken, varolan bir durumda bulunan ve güvenlik açığından yararlanılmasının önem derecesini azaltabilecek bir ayar, genel yapılandırma ya da en iyi uygulama anlamına gelir. Sizin durumunuzda aşağıdaki azaltıcı etkenler işinize yarayabilir:

-   Güvenlik açığından, e-posta yoluyla otomatik olarak yararlanılamaz. Saldırının başarılı olması için, kullanıcının e-posta iletisiyle gönderilen bir eki açması gerekir.
-   Web tabanlı saldırı senaryosunda, bir saldırganın bu açıktan yararlanmak için kullanılan bir Web sayfası içeren bir Web sitesi barındırabilir. Ayrıca, kullanıcı tarafından sağlanan içeriği veya reklamları kabul eden ya da barındıran güvenliği aşılmış Web sitelerinde, bu güvenlik açığından yararlanılabilecek özel hazırlanmış içerik bulunabilir. Ancak bu durumların hiçbirinde, saldırganın, bu Web sitelerini ziyaret etmek için kullanıcıyı zorlama yolu yoktur. Bunun yerine, saldırganın kullanıcıları bir e-posta iletisindeki veya Instant Messenger iletisindeki kendi Web sitesine götüren bir bağlantıyı tıklatmalarını sağlayarak onları bu Web sitesini ziyaret etmeye ikna etmesi gerekir.

#### XML Dış Varlıklarını Çözümleme Güvenlik Açığı - CVE-2011-1280 için Geçici Çözümler

Microsoft bu güvenlik açığı için herhangi bir geçici çözüm belirleyememiştir.

#### XML Dış Varlıklarını Çözümleme Güvenlik Açığı - CVE-2011-1280 Hakkında SSS

**Bu güvenlik açığının kapsamı nedir?**  
Bu, bir bilginin açığa çıkması güvenlik açığıdır. Bu güvenlik açığından başarıyla yararlanan bir saldırgan, hedef sistemde bulunan bir dosyadaki verileri okuyabilir. Bu güvenlik açığının, bir saldırganın doğrudan kod yürütmesine veya kullanıcı haklarını yükseltmesine izin vermeyeceğini unutmayın; ancak, etkilenen sistemden daha fazla yararlanmak amacıyla kullanılacak bilgileri üretmek için kullanılabilir.

**Güvenlik açığına ne neden olmaktadır?**  
Güvenlik açığı, XML dış varlıklarının diğer XML dış varlık bildirimleri içinde çözümlenmesi nedeniyle ortaya çıkar.

**XML Düzenleyicisi nedir?**  
XML Düzenleyicisi, XML dosyalarının düzenleyicisidir. Tek başına bir XML dosyasında veya Visual Studio projesiyle ilişkilendirilmiş bir dosyada kullanılabilir. Bir şema veya belge türü tanımı sağlanırsa, izin verilen öğeleri ve öznitelikleri listelemek üzere IntelliSense tarafından kullanılır. XML Düzenleyicisi şu dosya uzantılarıyla ilişkilendirilir: .config, .dtd, .xml, .xsd, .xdr, .xsl, .xslt ve .vssettings. XML Düzenleyicisi ayrıca belirli bir düzenleyici kaydettirilmemiş olan ve XML ya da DTD içeriği bulunan diğer dosya türleriyle de ilişkilendirilir. XML Düzenleyicisi hakkında daha fazla bilgi için, [XML Düzenleyicisi](http://msdn.microsoft.com/en-us/library/ms255810.aspx) adlı MSDN makalesine bakın.

**Web Hizmeti Bulma (.disco) Dosyası nedir?**  
.disco belgesi, Web Hizmeti'nin açıklandığı diğer kaynaklara bağlantılar içeren bir XML belgesidir ve okunabilir bir HTML dosyasına ya da arabirim sözleşmesini içeren bir WSDL dosyasına benzer.

**XML Dış Varlığı nedir?**  
XML belgesi bir veya birden çok depolama biriminden oluşabilir. Bunlara varlık adı verilir. Her biri içeriğe sahiptir ve bir varlık adıyla tanımlanır. Dış varlıklar bir XML belgesinin bir dış dosyaya başvurmasına olanak verir. Dış varlıklar metin veya ikili veri içerir. Metin içeriyorlarsa, dış dosyanın içeriği başvuru noktasına eklenir ve başvuruyu yapan belgenin bir parçası olarak ayrıştırılır. Daha fazla bilgi için belirtime bakın.

**Bir saldırgan bu güvenlik açığından yararlanarak neler yapabilir?**  
Bu güvenlik açığından başarıyla yararlanan bir saldırgan hedef sistemdeki dosyalardan veri okuyabilir. Bu güvenlik açığının, bir saldırganın doğrudan kod yürütmesine veya kullanıcı haklarını yükseltmesine izin vermeyeceğini unutmayın; ancak, etkilenen sistemden daha fazla yararlanmak amacıyla kullanılacak bilgileri üretmek için kullanılabilir.

**Bir saldırgan bu güvenlik açığından nasıl yararlanabilir?**  
Bir saldırganın bu güvenlik açığından yararlanabilmesi için, kullanıcının özel hazırlanmış bir Web Hizmeti Bulma (.disco) dosyasını Etkilenen Yazılımlar tablosunda listelenen uygulamalardan biriyle açması gerekir.

E-posta tabanlı saldırı senaryosunda, saldırgan kullanıcıya özel hazırlanmış bir Web Hizmeti Bulma (.disco) dosyası gönderip bu dosyayı Etkilenen Yazılımlar tablosunda listelenen uygulamalardan biriyle açmaya ikna ederek bu güvenlik açığından yararlanabilir.

Web tabanlı saldırı senaryosunda, saldırganın, bu açıktan yararlanma girişimi için kullanılan özel hazırlanmış Web Hizmeti Bulma (.disco) dosyasını içeren bir Web sitesini barındırması gerekir. Ayrıca, kullanıcı tarafından sağlanan içeriği kabul eden ya da barındıran güvenliği aşılmış Web sitelerinde, bu güvenlik açığından yararlanılabilecek özel hazırlanmış içerik bulunabilir. Saldırganın, bir kullanıcıyı özel hazırlanmış bir Web sitesini ziyaret etmeye zorlama yolu yoktur. Bunun yerine, saldırganın kullanıcıları kendi Web sitesine götüren bir bağlantıyı tıklatmaya ikna ederek onları bu siteye çekmesi ve daha sonra özel hazırlanmış Web Hizmeti Bulma (.disco) dosyasını açmalarını sağlaması gerekir.

**Bu güvenlik açığından birincil derecede hangi sistemler etkileniyor?**  
Bu güvenlik açığı, herhangi bir kötü amaçlı eylemin gerçekleşmesi için kullanıcının kötü amaçlı bir XML dosyasını etkilenen uygulamalardan biriyle açmasını gerektirir. Dolayısıyla, XML dosyalarının sıkça görüntülendiği sistemler (iş istasyonları veya terminal sunucuları gibi), bu açık karşısında en fazla risk altındadır.

**Güncelleştirme ne yapıyor?**  
Güncelleştirme, XML Düzenleyicisi'nin bir Web Hizmeti Bulma (.disco) dosyasındaki dış varlıkları çözümleme biçimini düzelterek bu güvenlik açığını giderir.
