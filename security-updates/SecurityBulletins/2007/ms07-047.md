Security Bulletin

Microsoft Güvenlik Bülteni MS07-047 - Önemli
============================================

Windows Media Player'daki Güvenlik Açıkları Uzaktan Kod Yürütülmesine İzin Verebilir (936782)
---------------------------------------------------------------------------------------------

Yayım Tarihi: 14 Ağustos 2007 Salı | Güncelleştirme Tarihi: 10 Eylül 2008 Çarşamba

**Sürüm:** 2.1

### Genel Bilgiler

#### Yürütmeyle İlgili Özet

Bu önemli güvenlik güncelleştirmesi, özel olarak bildirilen iki güvenlik açığını giderir. Bu güvenlik açıkları, kullanıcı Windows Media Player'da özel hazırlanmış bir dosyayı görüntülediğinde kod yürütülmesine izin verebilir. Hesapları, sistemde az sayıda kullanıcı hakkıyla yapılandırılmış olan kullanıcılar, yönetici haklarıyla çalışan kullanıcılardan daha az etkilenebilir.

Bu, desteklenen Windows Media Player 7.1, 9, 10 ve 11 sürümleri için önemli bir güvenlik güncelleştirmesidir. Daha fazla bilgi için, bu bölümdeki **Etkilenen ve Etkilenmeyen Yazılımlar** alt bölümüne bakın.

Bu güvenlik açıkları hakkında daha fazla bilgi için, bir sonraki bölüm olan **Güvenlik Açığı Bilgileri** altında, ilgili güvenlik açığı girdisinin Sık Sorulan Sorular (SSS) alt bölümüne bakın.

**Öneri.** Microsoft, müşterilerinin güncelleştirmeyi uygulamasını önerir.

**Bilinen Sorunlar.** [Microsoft Bilgi Bankası makalesi 936782](http://support.microsoft.com/kb/936782), bu güvenlik güncelleştirmesini yükleyen müşterilerin karşılaşabileceği bilinen sorunları belgelemektedir. Bu makale ayrıca, bu sorunlar için önerilen çözümleri de içermektedir.

#### Etkilenen ve Etkilenmeyen Yazılımlar

Burada listelenen yazılımlar, hangi sürümlerin etkilendiğini belirlemek amacıyla sınanmıştır. Diğer sürümler destek ömrünü tamamlamıştır veya bu açıktan etkilenmemektedir. Yazılımınızın sürümünün destek ömrünü belirlemek için [Microsoft Destek Ömrü](http://go.microsoft.com/fwlink/?linkid=21742) Web sitesini ziyaret edin.

Bu Güvenlik Güncelleştirmesi İle İlgili Sık Sorulan Sorular (SSS)
-----------------------------------------------------------------

<span></span>
**Bu güvenlik bülteni 10 Eylül 2008'de neden yeniden düzenlendi?**  
Microsoft bu güvenlik bültenini, Etkilenen Yazılımlar tablosundan Windows XP Service Pack 3 üzerinde Windows Media Player 9'u kaldırarak yeniden düzenledi. Bu, yalnızca bir bülten değişikliğidir; algılamada veya ikili dosyalarda değişiklik yapılmamıştır. Sistemlerini başarıyla güncelleştirmiş olan müşterilerin bu güncelleştirmeyi yeniden yüklemeleri gerekmez.

**Bu güvenlik bülteni 12 Ağustos 2008'de neden yeniden düzenlendi?**  
Microsoft bu güvenlik bültenini, Windows Media Player 9, Windows Media Player 10 ve Windows Media Player 11 çalıştırıldığında etkilenen yazılım olarak Windows XP Service Pack 3 eklenecek biçimde yeniden düzenledi. Bu, yalnızca bir algılama değişikliğidir; ikili dosyalarda değişiklik yapılmamıştır. Sistemlerini başarıyla güncelleştirmiş olan müşterilerin bu güncelleştirmeyi yeniden yüklemeleri gerekmez.

**Bu güvenlik güncelleştirmesini yükleyen müşterilerin karşılaşabileceği bilinen sorunlar nelerdir?**  
[Microsoft Bilgi Bankası makalesi 936782](http://support.microsoft.com/kb/936782), bu güvenlik güncelleştirmesini yükleyen müşterilerin karşılaşabileceği bilinen sorunları belgelemektedir. Bu makale ayrıca, bu sorunlar için önerilen çözümleri de içermektedir.

**Bu güvenlik bülteninde açıklanan yazılımın eski bir sürümünü kullanıyorum. Ne yapmalıyım?**  
Bu bültende listelenen etkilenen yazılımlar, hangi sürümlerinin etkilendiğini belirlemek amacıyla sınanmıştır. Diğer sürümler destek ömrünü tamamlamıştır. Yazılım sürümünüzün destek ömrünü belirlemek için [Microsoft Destek Ömrü](http://go.microsoft.com/fwlink/?linkid=21742) Web sitesini ziyaret edin.

Bu yazılımların eski sürümlerini kullanan müşterilerimiz, ortaya çıkabilecek güvenlik açıklarından etkilenme olasılığını ortadan kaldırmak için desteklenen sürümlere acilen yükseltmelidir. Windows Ürün Destek Ömrü hakkında daha fazla bilgi için, şu [Microsoft Destek Ömrü](http://go.microsoft.com/fwlink/?linkid=21742) Web sitesini ziyaret edin. Bu yazılım sürümleri için genişletilmiş güvenlik güncelleştirmesi destek süreci hakkında daha fazla bilgi için, [Microsoft Ürün Destek Hizmetleri Web sitesini](http://go.microsoft.com/fwlink/?linkid=33328) ziyaret edin.

Yazılımların eski sürümleri için ek özel destek isteyen müşteriler, Microsoft hesap ekibi temsilcilerine, Teknik Hesap Yöneticisi'ne veya özel destek seçenekleri için uygun Microsoft ortağı temsilcisine başvurmalıdır. Alliance, Premier veya Authorized sözleşme sahibi olmayan müşteriler yerel Microsoft satış ofisine de başvurabilir. Başvuru bilgileri için, [Microsoft Worldwide Information Web sitesini](http://go.microsoft.com/fwlink/?linkid=33329) ziyaret edin, bulunduğunuz ülkeyi seçin ve **Go**'yu (Git) tıklatıp telefon numaralarının listesini görüntüleyin. Aradığınızda, yerel Premier Destek satış yöneticisiyle görüşmek istediğinizi söyleyin. Daha fazla bilgi için, bkz: [Windows İşletim Sistemlerinin Ürün Destek Ömrü Hakkında SSS](http://go.microsoft.com/fwlink/?linkid=33330).

**Bu güncelleştirme, işlevlerde güvenlikle ilgili değişiklikler içeriyor mu?**  
Evet. Bu bültenin Güvenlik Açığı Bilgileri bölümünün altında ilgili güvenlik açığı girdisi için Sık Sorulan Sorular (SSS) alt bölümünde listelenen değişikliklerin yanı sıra, bu güncelleştirme, Windows Media Player'da kapsamlı koruma değişiklikleri de içerir. Kapsamlı koruma hakkında daha fazla bilgi için bkz: [Microsoft Bilgi Bankası makalesi 940893](http://support.microsoft.com/kb/940893).

Windows Media Player Dış Görünümleri Ayrıştırma Kod Yürütme Güvenlik Açığı – CVE-2007-3037  
------------------------------------------------------------------------------------------
  
<span></span>
Windows Media Player'ın dış görünümleri ayrıştırmasında bir kod yürütme güvenlik açığı bulunmaktadır. Bu açığı başarıyla kullanan bir saldırgan, etkilenen sistemin tüm denetimini ele geçirebilir.
  
Bu güvenlik açığını Ortak Güvenlik Açıkları ve Etkilenmeler listesinde standart girdi olarak görüntülemek için, bkz: [CVE-2007-3037](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2007-3037).
  
#### Windows Media Player Dış Görünümleri Ayrıştırma Kod Yürütme Güvenlik Açığı – CVE-2007-3037 için Azaltıcı Etkenler
  
Azaltıcı etken, varolan bir durumda bulunan ve güvenlik açığından yararlanılmasının önem derecesini azaltabilecek bir ayar, genel yapılandırma ya da en iyi uygulama anlamına gelir. Sizin durumunuzda aşağıdaki azaltıcı etkenler işinize yarayabilir:
  
-   Kullanıcı bir Windows Media Player dış görünüm dosyasını yükleme girişiminde bulunduğunda, dış görünüm uygulanmadan önce kullanıcıya bir "Windows Media'yı Karşıdan Yükle" iletişim kutusu görüntülenir.  
-   WMZ ve WMD dosyalarını kullanarak güvenlik açığından yararlanma girişimleri, kullanıcının dış görünümü karşıdan yükledikten sonra görüntülemesini veya uygulamasını gerektirir.
  
#### Windows Media Player Dış Görünümleri Ayrıştırma Kod Yürütme Güvenlik Açığı – CVE-2007-3037 için Geçici Çözümler
  
**WMZ ve WMD dosya uzantılarının ilişkilendirmesini kaldırma**
  
Windows'da WMZ ve WMD dosya ilişkilendirmelerinin kaldırılması Windows Media Player'da WMZ ve WMD dosyalarının önizlenmesini veya açılmasını engeller.
  
1.  **Başlat**'ı tıklatın, **Çalıştır**'ı tıklatın, "**regedit**" (tırnak işaretleri olmadan) yazın ve Tamam'ı tıklatın.  
2.  **HKEY\_CLASSES\_ROOT**'u genişletin, .**WMZ**'yi tıklatın ve sonra sağ tıklatıp **Ver**'i seçin. Not: Bu işlem, varsayılan olarak 'Belgelerim' klasöründe bu kayıt defteri anahtarının bir yedeğini oluşturur.  
3.  **Kayıt Defteri Dosyası Ver** penceresinde, "WMZ dosyası ilişkilendirmesi kayıt defteri yedeği.reg" yazın ve **Kaydet**'e basın.  
    Bu işlem, varsayılan olarak "Belgelerim" klasöründe bu kayıt defteri anahtarının bir yedeğini oluşturur.  
4.  Klavyede **Delete** tuşuna basarak kayıt defteri anahtarını silin. Kayıt defteri anahtarının silinmesini onaylamak için **Evet**'i seçin.  
5.  2-5 arası adımları **WMD** anahtarı için yineleyin.
  
**Geçici Çözümün Etkisi**: Bu geçici çözüm, kullanıcıların dış görünüm dosyalarını çift tıklatarak Windows Media Player'a uygulamalarını engeller. Kullanıcılar varsayılan 'skins' dizinindeki dış görünüm dosyalarını uygulayabilir.
  
**Wmp.dll dosyasının kaydını kaldırma**
  
1.  Wmp.dll kayıt defteri anahtarının kaydını kaldırmak, etkilenen sistemin bu açıktan yararlanma girişimlerine karşı korunmasına yardımcı olur. Wmp.dll kayıt defteri anahtarını değiştirmek için aşağıdaki adımları izleyin.  
2.  **Başlat**'ı ve sonra **Çalıştır**'ı tıklatın, "**regsvr32 -u %windir%\\system32\\wmp.dll**" (tırnak işaretleri olmadan) yazın ve **Tamam**'ı tıklatın.  
3.  İşlemin başarılı olduğunu bildiren bir iletişim kutusu görüntülendiğinde, **Tamam**'ı tıklatın.  
4.  **Dosya Türleri** sekmesini seçin.
  
**Geçici Çözümün Etkisi**: Bu geçici çözüm, Windows Media Player'ı ve katıştırılmış Windows Media ActiveX Denetimi'ni kullanan uygulamaları devre dışı bırakır.
  
#### Windows Media Player Dış Görünümleri Ayrıştırma Kod Yürütme Güvenlik Açığı – CVE-2007-3037 Hakkında SSS
  
**Bu güvenlik açığının kapsamı nedir? **  
Windows Media Player'ın dış görünümleri ayrıştırmasında bir kod yürütme güvenlik açığı bulunmaktadır. Bu açığı başarıyla kullanan bir saldırgan, etkilenen sistemin tüm denetimini ele geçirebilir. Böylece saldırgan program yükleyebilir ya da verileri görüntüleyebilir, değiştirebilir veya silebilir.
  
**Güvenlik açığına ne neden olmaktadır? **  
Windows Media Player, dış görünüm dosyalarındaki üstbilgi bilgilerini yanlış şekilde işlemektedir.
  
**Dış görünüm dosyası nedir? **  
Dış görünümler, Windows Media Player'da yeni bir görünüm oluşturmayı sağlayan ve komut dosyası, çizim, medya ve metin dosyalarından oluşan bir kümedir. Dış görünümleri kullanarak Windows Media Player'ın yalnızca görünüşünü değiş çalışmasını da değiştirebilirsiniz. Windows Media Player dış görünüm dosyaları WMZ ve WMD dosyaları olarak dağıtılır.
  
**Bir saldırgan bu güvenlik açığından yararlanarak neler yapabilir? **  
Bu açığı başarıyla kullanan bir saldırgan, etkilenen sistemin tüm denetimini ele geçirebilir. Konuk işletim sisteminde hesapları daha az kullanıcı hakkına sahip olacak şekilde yapılandırılmış olan kullanıcılar, konuk işletim sisteminde yönetici haklarıyla çalışan kullanıcılara göre daha az etkilenir.
  
**Bir saldırgan güvenlik açığından nasıl yararlanabilir? **  
Web tabanlı saldırı senaryosunda, bir saldırgan Windows Media Player üzerinden bu güvenlik açığından yararlanmak üzere özel hazırlanmış bir dış görünüm dosyası barındırabilir ve sonra kullanıcıyı dış görünüm dosyasını görüntülemeye ikna edebilir. Ancak saldırganın, bu Web sitelerini ziyaret etmek için kullanıcıyı zorlama yolu yoktur. Bunun yerine, saldırganın kullanıcıları bir e-posta iletisindeki veya Anlık İleti isteğindeki kendi Web sitesine götüren bir bağlantıyı tıklatmalarını sağlayarak onları bu Web sitesine çekmesi gerekir.
  
E-posta tabanlı saldırı senaryosunda, saldırgan kullanıcıya özel hazırlanmış bir dış görünüm dosyası göndererek ve bu dosyayı açmaya ikna ederek güvenlik açığından yararlanabilir.
  
**Bu güvenlik açığından birincil derecede hangi sistemler etkileniyor? **  
Bu güvenlik açığında, herhangi bir kötü amaçlı eylemin gerçekleşmesi için kullanıcının oturum açmış olması ve Windows Media Player'da kötü amaçlı dış görünüm dosyasını açması gerekir. Dolayısıyla, Windows Media Player'ın sıkça kullanıldığı sistemler (iş istasyonları veya terminal sunucuları gibi), bu açık karşısında en fazla risk altındadır.
  
**Güncelleştirme ne yapıyor? **  
Güncelleştirme, Windows Media Player'da dış görünüm dosyalarındaki üstbilgi bilgileri doğru olarak işlenecek şekilde değişiklik yaparak güvenlik açığını ortadan kaldırır.
  
**Bu güvenlik bülteni yayımlandığında bu güvenlik açığı genel olarak duyurulmuş muydu? **  
Hayır. Microsoft, bu güvenlik açığı hakkında bilgilere sorumlu kişiler aracılığıyla ulaştı. Bu güvenlik bülteni ilk yayımlandığında, Microsoft bu güvenlik açığının genel olarak duyurulduğuna dair herhangi bir bilgi almamıştı.

<span></span>
Windows Media Player'da, bu açıktan başarıyla yararlanan bir saldırganın etkilenen sistemin denetimini tümüyle ele geçirmesine olanak verebilen bir uzaktan kod yürütme güvenlik açığı bulunmaktadır.
  
Bu güvenlik açığını Ortak Güvenlik Açıkları ve Etkilenmeler listesinde standart girdi olarak görüntülemek için, bkz: [CVE-2007-3035](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2007-3035).
  
#### Windows Media Player Dış Görünümleri Açma Kod Yürütme Güvenlik Açığı – CVE-2007-3035 için Azaltıcı Etkenler
  
Azaltıcı etken, varolan bir durumda bulunan ve güvenlik açığından yararlanılmasının önem derecesini azaltabilecek bir ayar, genel yapılandırma ya da en iyi uygulama anlamına gelir. Sizin durumunuzda aşağıdaki azaltıcı etkenler işinize yarayabilir:
  
-   Kullanıcı bir Windows Media Player dış görünüm dosyasını yükleme girişiminde bulunduğunda, dış görünüm uygulanmadan önce kullanıcıya bir "Windows Media'yı Karşıdan Yükle" iletişim kutusu görüntülenir.  
-   WMZ ve WMD dosyalarını kullanarak güvenlik açığından yararlanma girişimleri, kullanıcının dış görünümü karşıdan yükledikten sonra görüntülemesini veya uygulamasını gerektirir.
  
#### Windows Media Player Dış Görünümleri Açma Kod Yürütme Güvenlik Açığı – CVE-2007-3035 için Geçici Çözümler
  
**WMZ ve WMD dosya uzantılarının ilişkilendirmesini kaldırma**
  
Windows'da WMZ ve WMD dosya ilişkilendirmelerinin kaldırılması Windows Media Player'da WMZ ve WMD dosyalarının önizlenmesini veya açılmasını engeller.
  
1.  **Başlat**'ı tıklatın, **Çalıştır**'ı tıklatın, "**regedit**" (tırnak işaretleri olmadan) yazın ve Tamam'ı tıklatın.  
2.  **HKEY\_CLASSES\_ROOT**'u genişletin, .**WMZ**'yi tıklatın ve sonra sağ tıklatıp **Ver**'i seçin. Not: Bu işlem, varsayılan olarak 'Belgelerim' klasöründe bu kayıt defteri anahtarının bir yedeğini oluşturur.  
3.  **Kayıt Defteri Dosyası Ver** penceresinde, "WMZ dosyası ilişkilendirmesi kayıt defteri yedeği.reg" yazın ve **Kaydet**'e basın.  
    Bu işlem, varsayılan olarak "Belgelerim" klasöründe bu kayıt defteri anahtarının bir yedeğini oluşturur.  
4.  Klavyede **Delete** tuşuna basarak kayıt defteri anahtarını silin. Kayıt defteri anahtarının silinmesini onaylamak için **Evet**'i seçin.  
5.  2-5 arası adımları **WMD** anahtarı için yineleyin.
  
**Geçici Çözümün Etkisi**: Bu geçici çözüm, kullanıcıların dış görünüm dosyalarını çift tıklatarak Windows Media Player'a uygulamalarını engeller. Kullanıcılar varsayılan 'skins' dizinindeki dış görünüm dosyalarını uygulayabilir.
  
**Wmp.dll dosyasının kaydını kaldırma**
  
1.  Wmp.dll kayıt defteri anahtarının kaydını kaldırmak, etkilenen sistemin bu açıktan yararlanma girişimlerine karşı korunmasına yardımcı olur. Wmp.dll kayıt defteri anahtarını değiştirmek için aşağıdaki adımları izleyin.  
2.  **Başlat**'ı ve sonra **Çalıştır**'ı tıklatın, "**regsvr32 -u %windir%\\system32\\wmp.dll**" (tırnak işaretleri olmadan) yazın ve **Tamam**'ı tıklatın.  
3.  İşlemin başarılı olduğunu bildiren bir iletişim kutusu görüntülendiğinde, **Tamam**'ı tıklatın.  
4.  **Dosya Türleri** sekmesini seçin.
  
**Geçici Çözümün Etkisi**: Bu geçici çözüm, Windows Media Player'ı ve katıştırılmış Windows Media ActiveX Denetimi'ni kullanan uygulamaları devre dışı bırakır.
  
#### Windows Media Player Dış Görünümleri Açma Kod Yürütme Güvenlik Açığı – CVE-2007-3035 Hakkında SSS
  
**Bu güvenlik açığının kapsamı nedir? **  
Windows Media Player'ın dış görünümleri ayrıştırmasında bir kod yürütme güvenlik açığı bulunmaktadır. Bu açığı başarıyla kullanan bir saldırgan, etkilenen sistemin tüm denetimini ele geçirebilir. Böylece saldırgan program yükleyebilir ya da verileri görüntüleyebilir, değiştirebilir veya silebilir.
  
**Güvenlik açığına ne neden olmaktadır? **  
Windows Media Player, dış görünüm dosyalarındaki üstbilgi bilgilerini yanlış şekilde işlemektedir.
  
**Dış görünüm dosyası nedir? **  
Dış görünümler, Windows Media Player'da yeni bir görünüm oluşturmayı sağlayan ve komut dosyası, çizim, medya ve metin dosyalarından oluşan bir kümedir. Dış görünümleri kullanarak Windows Media Player'ın yalnızca görünüşünü değiş çalışmasını da değiştirebilirsiniz. Windows Media Player dış görünüm dosyaları WMZ ve WMD dosyaları olarak dağıtılır.
  
**Bir saldırgan bu güvenlik açığından yararlanarak neler yapabilir? **  
Bu açığı başarıyla kullanan bir saldırgan, etkilenen sistemin tüm denetimini ele geçirebilir. Konuk işletim sisteminde hesapları daha az kullanıcı hakkına sahip olacak şekilde yapılandırılmış olan kullanıcılar, konuk işletim sisteminde yönetici haklarıyla çalışan kullanıcılara göre daha az etkilenir.
  
**Bir saldırgan güvenlik açığından nasıl yararlanabilir? **  
Web tabanlı saldırı senaryosunda, bir saldırgan Windows Media Player üzerinden bu güvenlik açığından yararlanmak üzere özel hazırlanmış bir dış görünüm dosyası barındırabilir ve sonra kullanıcıyı dış görünüm dosyasını görüntülemeye ikna edebilir. Ancak saldırganın, bu Web sitelerini ziyaret etmek için kullanıcıyı zorlama yolu yoktur. Bunun yerine, saldırganın kullanıcıları bir e-posta iletisindeki veya Anlık İleti isteğindeki kendi Web sitesine götüren bir bağlantıyı tıklatmalarını sağlayarak onları bu Web sitesine çekmesi gerekir.
  
E-posta tabanlı saldırı senaryosunda, saldırgan kullanıcıya özel hazırlanmış bir dış görünüm dosyası göndererek ve bu dosyayı açmaya ikna ederek güvenlik açığından yararlanabilir.
  
**Bu güvenlik açığından birincil derecede hangi sistemler etkileniyor? **  
Bu güvenlik açığında, herhangi bir kötü amaçlı eylemin gerçekleşmesi için kullanıcının oturum açmış olması ve Windows Media Player'da kötü amaçlı dış görünüm dosyasını açması gerekir. Dolayısıyla, Windows Media Player'ın sıkça kullanıldığı sistemler (iş istasyonları veya terminal sunucuları gibi), bu açık karşısında en fazla risk altındadır.
  
**Güncelleştirme ne yapıyor? **  
Güncelleştirme, Windows Media Player'da dış görünüm dosyalarındaki üstbilgi bilgileri doğru olarak işlenecek şekilde değişiklik yaparak güvenlik açığını ortadan kaldırır.
  
**Bu güvenlik bülteni yayımlandığında bu güvenlik açığı genel olarak duyurulmuş muydu? **  
Hayır. Microsoft, bu güvenlik açığı hakkında bilgilere sorumlu kişiler aracılığıyla ulaştı. Bu güvenlik bülteni ilk yayımlandığında, Microsoft bu güvenlik açığının genel olarak duyurulduğuna dair herhangi bir bilgi almamıştı.

    3.  **Dosya adının tümü veya bir parçası** kutusuna, uygun dosya bilgileri tablosundan bir dosya adı yazın ve **Ara**'yı tıklatın.
