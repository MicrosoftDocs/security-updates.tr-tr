Security Bulletin

Microsoft Güvenlik Bülteni MS09-051 - Kritik
============================================

Windows Media Çalışma Zamanı Modülü'ndeki Güvenlik Açıkları Uzaktan Kod Yürütülmesine İzin Verebilir (975682)
-------------------------------------------------------------------------------------------------------------

Yayım Tarihi: 13 Ekim 2009 Salı | Güncelleştirme Tarihi: 10 Kasım 2009 Salı

**Sürüm:** 2.0

### Genel Bilgiler

#### Yürütmeyle İlgili Özet

Bu güvenlik güncelleştirmesi Windows Media Çalışma Zamanı Modülü'ndeki özel olarak bildirilen iki güvenlik açığını giderir. Bu güvenlik açıkları, bir kullanıcı özel hazırlanmış bir medya dosyasını açarsa veya Web içeriği sağlayan bir Web sitesinden ya da başka bir uygulamadan özel hazırlanmış akış içeriği alırsa uzaktan kod yürütülmesine izin verebilir. Bu güvenlik açıklarından başarıyla yararlanan bir saldırgan, yerel kullanıcı ile aynı haklara sahip olabilir. Hesapları, sistemde az sayıda kullanıcı hakkıyla yapılandırılmış olan kullanıcılar, yönetici haklarıyla çalışan kullanıcılardan daha az etkilenebilir.

Bu güvenlik güncelleştirmesi Microsoft Windows 2000, Windows XP, Itanium tabanlı sürümleri dışında Windows Server 2003, Windows Vista ve Itanium tabanlı sürümleri dışında Windows Server 2008'in tüm desteklenen sürümleri üzerinde DirectShow WMA Ses Codec Bileşeni, Windows Media Ses Kodu Çözücü ve Ses Sıkıştırma Yöneticisi için Kritik olarak derecelendirilmiştir. Daha fazla bilgi için, bu bölümdeki **Etkilenen ve Etkilenmeyen Yazılımlar** alt bölümüne bakın.

Güvenlik güncelleştirmesi, Windows Media Çalışma Zamanı Modülü'nün ASF dosyalarını işleme ve sıkıştırılmış ses dosyalarındaki işlevleri başlatma biçimini değiştirerek bu güvenlik açıklarını ortadan kaldırır. Bu güvenlik açıkları hakkında daha fazla bilgi için, bir sonraki bölüm olan **Güvenlik Açığı Bilgileri** altında, ilgili güvenlik açığı girdisinin Sık Sorulan Sorular (SSS) alt bölümüne bakın.

**Öneri.** Müşterilerin büyük bir bölümü otomatik güncelleştirmeleri etkinleştirmiş durumdadır ve bu güvenlik güncelleştirmesi otomatik olarak karşıdan yüklenip kurulacağı için herhangi bir işlem yapmaları gerekmemektedir. Otomatik güncelleştirmeleri etkinleştirmemiş olan müşterilerin güncelleştirmeleri el ile denetlemeleri ve bu güncelleştirmeyi yüklemeleri gerekir. Otomatik güncelleştirmelerin özel yapılandırma seçenekleri hakkında bilgi için, bkz. [Microsoft Bilgi Bankası makalesi 294871](http://support.microsoft.com/kb/294871).

Yöneticiler ve kurumsal yüklemeler ya da bu güvenlik güncelleştirmesini el ile yüklemek isteyen son kullanıcılar için, Microsoft, müşterilerinin güncelleştirme yönetim yazılımlarını kullanarak veya [Microsoft Update](http://go.microsoft.com/fwlink/?linkid=40747) hizmeti aracılığıyla güncelleştirmeleri denetleyerek güncelleştirmeyi hemen uygulamalarını önerir.

Bu bültenin sonrasında yer alan **Algılama ve Dağıtım Araçları ve Kılavuzu** bölümüne de bakın.

**Bilinen Sorunlar.** Yok

Bu Güvenlik Güncelleştirmesi İle İlgili Sık Sorulan Sorular (SSS)  
-----------------------------------------------------------------
  
<span></span>
**Bu bülten 10 Kasım 2009'da neden yeniden yayımlandı?**    
Microsoft bu bülteni, bir algılama sorununu gidermek için Microsoft Windows 2000 Service Pack 4 üzerinde Ses Sıkıştırma Yöneticisi güncelleştirmesinin yeniden sunulduğunu duyurmak üzere yeniden yayımladı. Bu, yalnızca bir algılama değişikliğidir; ikili dosyalarda değişiklik yapılmamıştır. Sistemlerini başarıyla güncelleştirmiş olan müşterilerin bu güncelleştirmeyi yeniden yüklemeleri gerekmez.
  
**Dosya bilgileri ayrıntılarını nerede bulabilirim?**    
Dosya bilgileri ayrıntılarının konumu için **Güvenlik Güncelleştirmesi Dağıtımı** bölümündeki başvuru tablolarına bakın.
  
**Windows Media Player'daki Güvenlik Açığı Uzaktan Kod Yürütülmesine İzin Verebilir adlı MS09-052 ile bu bülten arasında nasıl bir ilişki vardır?**    
Bu bülten, Windows Media Player'daki Güvenlik Açığı Uzaktan Kod Yürütülmesine İzin Verebilir adlı [MS09-052](http://go.microsoft.com/fwlink/?linkid=163913) ile ilişkili değildir. Kullanıcılar tam olarak korunduklarından emin olmak için her iki güncelleştirmeyi de yüklemelidir.
  
**Sistemime Windows Media Player yüklenmiş durumda. Bu güncelleştirmelerin bir bölümü neden sunulmuyor? **  
Bu güncelleştirmeler yalnızca Windows Media Player'ın yanı sıra bu güvenlik açıklarından etkilenen isteğe bağlı ses codec bileşenlerinin yüklü olduğu sistemlere otomatik olarak sunulur. KB969878 ile ilişkili codec bileşenleri yalnızca 32-bit sistemlere uygulanabilir. KB954155 ile ilişkili codec bileşenleri varsayılan olarak x64 tabanlı Windows Media Format SDK üzerinde yüklü değildir, ancak Windows Media Player aracılığıyla otomatik olarak veya [Microsoft Yükleme Merkezi](http://www.microsoft.com/downloads/en/default.aspx)'nden el ile karşıdan yüklenebilirler. Bu codec bileşenleri yüklü değilse ve güncelleştirmeyi elle yüklemeyi denerseniz, aşağıdaki hata iletilerinden birini alabilirsiniz:
  
-   Bu güncelleştirmeyi yüklemek için Windows Media Player 9 gerekmektedir.  
-   Bu güncelleştirmenin yüklenebilmesi için makinenizde Windows Media Format 9.5 SDK olmalıdır.
  
Bu iletiyi alırsanız, bu güncelleştirme kullandığınız sisteme uygulanamamaktadır.  Bu iletiler yalnızca güncelleştirmeyi el ile yüklemeyi denerseniz görüntülenir.
  
**Bu güncelleştirme neden bildirilen birkaç güvenlik açığını ele alıyor?**    
Bu güncelleştirme, bazı güvenlik açıklarını ortadan kaldırmaya yönelik değişikliklerin ilgili dosyalarda bulunması nedeniyle bu sorunlar için destek içermektedir. Müşterilerimiz birbiriyle neredeyse aynı birkaç güncelleştirme yerine, yalnızca bu güncelleştirmeyi yüklemelidir.
  
**Windows Media Format Software Developers Kit (SDK) nedir?**    
Windows Media Format SDK, uygulama geliştiricilerin Windows Media Format bileşenlerine erişebilmesini sağlar. Windows Media Format SDK'nin temel amacı geliştiricilerin Gelişmiş Sistemler Biçimi (ASF) dosyaları ve ağ akışları oynatan, yazan, düzenleyen, şifreleyen ve sunan uygulamalar oluşturabilmelerini sağlamaktır. Bu dosyalarda ve akışlarda genelde Windows Media Ses ve Video codec bileşenleri kullanılarak kodlanmış ses ve video içeriği bulunur. Ancak ASF her türlü veri içerebilir. Daha fazla bilgi için, [Windows Media Format SDK Teknoloji Sayfası](http://msdn.microsoft.com/en-us/windows/bb190326.aspx)'na bakın.
  
**Bu güvenlik bülteninde açıklanan yazılımın eski bir sürümünü kullanıyorum. Ne yapmalıyım?**    
Bu bültende listelenen etkilenen yazılımlar, hangi sürümlerinin etkilendiğini belirlemek amacıyla sınanmıştır. Diğer sürümler destek ömrünü tamamlamıştır. Yazılım sürümünüzün destek ömrünü belirlemek için [Microsoft Destek Ömrü](http://go.microsoft.com/fwlink/?linkid=21742) Web sitesini ziyaret edin.

Windows Ürün Destek Ömrü hakkında daha fazla bilgi için [Microsoft Destek Ömrü](http://go.microsoft.com/fwlink/?linkid=21742) Web sitesini ziyaret edin. Windows Media Çalışma Zamanı Modülü'nde Ses Örnekleme Hızı Güvenlik Açığı - CVE-2009-0555  
-----------------------------------------------------------------------------------------
  
<span></span>
Windows Media Player'da, özel olarak hazırlanan Gelişmiş Sistemler Biçimi (ASF) dosyalarının hatalı işlenmesi nedeniyle bir uzaktan kod yürütme güvenlik açığı bulunmaktadır. Saldırgan, Windows Media Player'ın etkilenen bir sürümü kullanılarak oynatıldığında uzaktan kod yürütülmesine olanak verecek şekilde özel olarak hazırlanmış bir ses dosyası oluşturarak bu güvenlik açığından yararlanabilir. Bu açığı başarıyla kullanan bir saldırgan, etkilenen sistemin tüm denetimini ele geçirebilir.
  
Bu güvenlik açığını Ortak Güvenlik Açıkları ve Etkilenmeler listesinde standart girdi olarak görüntülemek için, bkz: [CVE-2009-0555](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2009-0555).
  
#### Windows Media Çalışma Zamanı Modülü'nde Ses Örnekleme Hızı Güvenlik Açığı - CVE-2009-0555 için Azaltıcı Etkenler
  
Azaltıcı etken, varolan bir durumda bulunan ve güvenlik açığından yararlanılmasının önem derecesini azaltabilecek bir ayar, genel yapılandırma ya da en iyi uygulama anlamına gelir. Sizin durumunuzda aşağıdaki azaltıcı etkenler işinize yarayabilir:
  
-   Web tabanlı saldırı senaryosunda, bir saldırganın bu açıktan yararlanmak için kullanılan bir Web sayfası içeren bir Web sitesi barındırabilir. Ayrıca, kullanıcı tarafından sağlanan içeriği veya reklamları kabul eden ya da barındıran güvenliği aşılmış Web sitelerinde, bu güvenlik açığından yararlanılabilecek özel hazırlanmış içerik bulunabilir. Ancak bu durumların hiçbirinde, saldırganın, bu Web sitelerini ziyaret etmek için kullanıcıyı zorlama yolu yoktur. Bunun yerine, saldırganın kullanıcıları bir e-posta iletisindeki veya Instant Messenger iletisindeki kendi Web sitesine götüren bir bağlantıyı tıklatmalarını sağlayarak onları bu Web sitesine çekmesi gerekir.  
-   Bu güvenlik açığından başarıyla yararlanan bir saldırgan, yerel kullanıcı ile aynı haklara sahip olabilir. Hesapları, sistemde az sayıda kullanıcı hakkıyla yapılandırılmış olan kullanıcılar, yönetici haklarıyla çalışan kullanıcılardan daha az etkilenebilir.  
-   İsteğe bağlı Masaüstü Deneyimi bileşeni de yüklü olmadıkça Windows Server 2008 bu açıktan etkilenmez.
  
#### Windows Media Çalışma Zamanı Modülü'nde Ses Örnekleme Hızı Güvenlik Açığı - CVE-2009-0555 için Geçici Çözümler
  
Geçici çözüm, arka plandaki güvenlik açığını gidermeyen ama güvenlik açığı uygulanmadan önceki bilinen saldırı vektörlerinin engellenmesine yardımcı olan bir ayar veya yapılandırma değişikliği anlamına gelir. Microsoft aşağıdaki geçici çözümleri sınamıştır ve bir geçici çözümün işlevselliği azaltıp azaltmadığı ilgili açıklamada belirtilmektedir:
  
-   **Wmspdmod.dll kaydını kaldırın**
  
    Bu DLL dosyasının kaydını kaldırmak için, ayrıcalığı yükseltilmiş bir komut isteminde şu komutu çalıştırın.
  
    32-bit Windows sistemler için:
  
    `Regsvr32.exe –u %WINDIR%\system32\wmspdmod.dll`
  
    x64 tabanlı Windows sistemleri:
  
    `Regsvr32.exe –u %WINDIR%\syswow64\wmspdmod.dll`
  
    **Geçici çözümün etkisi.** Windows Media Speech Code kullanan ASF dosyalarındaki ses akışları oynatılamaz.
  
    **Geçici çözüm nasıl geri alınır.**
  
    Ayrıcalığı yükseltilmiş bir komut isteminde aşağıdaki komutu çalıştırın.
  
    32-bit Windows sistemler için:
  
    `Regsvr32.exe %WINDIR%\system32\wmspdmod.dll`
  
    x64 tabanlı Windows sistemleri:
  
    `Regsvr32.exe %WINDIR%\syswow64\wmspdmod.dll`
  
#### Windows Media Çalışma Zamanı Modülü'nde Ses Örnekleme Hızı Güvenlik Açığı - CVE-2009-0555 Hakkında SSS
  
**Bu güvenlik açığının kapsamı nedir?**    
Bu bir uzaktan kod yürütme güvenlik açığıdır. Kullanıcı yönetimsel haklarla oturum açtıysa, bu güvenlik açığından başarıyla yararlanan bir saldırgan etkilenen sistemin denetimini tümüyle ele geçirebilir. Saldırgan, program yükleyebilir; verileri görüntüleyebilir, değiştirebilir veya silebilir; tüm kullanıcı haklarına sahip olan yeni hesaplar oluşturabilir. Hesapları, sistemde az sayıda kullanıcı hakkıyla yapılandırılmış olan kullanıcılar, yönetici haklarıyla çalışan kullanıcılardan daha az etkilenebilir.
  
**Güvenlik açığına ne neden olmaktadır?**    
Windows Media Çalışma Zamanı Modülü, Windows Media Speech codec bileşenini kullanan özel hazırlanmış ASF dosyalarını hatalı biçimde işlemektedir.
  
**Windows Media Format Çalışma Zamanı Modülü nedir?**    
Microsoft Windows Media Biçimi Çalışma Zamanı Modülü, Windows Media içeriğini kullanan uygulamalar için gerekli bilgi ve araçları sağlar.
  
**Gelişmiş Sistemler Biçimi (ASF) nedir?**    
ASF (Gelişmiş Sistemler Biçimi), ses ve video bilgilerini depolayan ve Internet gibi ağlar üzerinden çalışacak biçimde tasarlanan bir dosya biçimidir. Ses ve video akışı, slayt gösterileri ve eşzamanlı olaylar içerebilen bir sıkıştırılmış biçimdir. ASF, içeriğin sürekli veri akışı olarak teslim edilebilmesini sağlar. ASF dosyaları ASF, WMV veya WMA dosya uzantılı olabilir.
  
ASF, ses dosyasının arka planındaki biçimdir. ASF dosyaları birçok farklı dosya uzantısına sahip olabilir, en çok WMA ve WMV uzantıları kullanılır. ASF dosyaları, gerçek dosya biçimini yansıtmayan MP3 gibi diğer dosya uzantılarıyla da sunulabilir. Windows Media Player asıl dosya biçimini çözümler ve dosyayı yine de oynatmayı dener.
  
**Windows Media Ses Kodu Çözücü nedir?**    
Windows Media Ses Kodu Çözücü, [Windows Media Ses Kodlayıcı](http://msdn.microsoft.com/en-us/library/dd756600(vs.85).aspx) tarafından kodlanmış akışların kodunu çözer. Windows Media Audio Voice adlı codec bileşeni, konuşma içeren ses için en iyi duruma getirilmiş, düşük bit hızında sıkıştırma sağlar. Codec bileşeninin bu şekilde küçük örnekler oluşturmasının nedeni insan sesinin sınırlı frekans aralığıdır. En iyi duruma getirme işlemi, adanmış bir ses kodlayıcının müzik gibi daha karmaşık seslerin bulunduğu içerik için düşük kalitede çıkış sağladığı anlamına gelir. Ancak, Windows Media Audio Voice codec bileşeni insan sesi, müzik ve karma içerik için ayrı modlar sağlayarak olası bir kalite sorununu telafi eder. Codec bileşeni karma içeriği çözümleyerek, dosyanın her bölümü için hangi modun kullanılacağını belirler. Windows Media Ses Kodu Çözücü hakkında daha fazla bilgi için, [Windows Media Ses Kodu Çözücü'yü Kullanma](http://msdn.microsoft.com/en-us/library/dd443456(vs.85).aspx) adlı MSDN makalesine bakın.
  
**Bir saldırgan bu güvenlik açığından yararlanarak neler yapabilir?**    
Bu güvenlik açığından başarıyla yararlanan bir saldırgan rasgele kod çalıştırabilir. Saldırgan, program yükleyebilir; verileri görüntüleyebilir, değiştirebilir veya silebilir; tüm kullanıcı haklarına sahip olan yeni hesaplar oluşturabilir.
  
**Bir saldırgan güvenlik açığından nasıl yararlanabilir?**    
Bu güvenlik güncelleştirmesi, kullanıcının Windows Media Player'ın etkilenen sürümünde özel hazırlanmış bir ASF dosyasını açmasını gerektirir.
  
Web tabanlı saldırı senaryosunda, saldırganın, bu açıktan yararlanma girişimi için kullanılan özel hazırlanmış bir ASF dosyası barındırması gerekir. Saldırganın, bir kullanıcıyı özel hazırlanmış bir Web sitesini ziyaret etmeye zorlama yolu yoktur. Bunun yerine, saldırganın kullanıcıları bir e-posta iletisindeki veya Instant Messenger iletisindeki kendi Web sitesine götüren bir bağlantıyı tıklatmaya ikna ederek onları bu siteye çekmesi gerekir.
  
E-posta tabanlı saldırı senaryosunda, saldırgan kullanıcıya özel hazırlanmış bir ASF dosyasının katıştırılmış olduğu bir medya dosyası gönderip kullanıcıyı bu belgeyi açmaya ikna ederek bu güvenlik açığından yararlanabilir.
  
**Bu güvenlik açığından birincil derecede hangi sistemler etkileniyor?**    
İş istasyonları veya terminal sunucuları gibi, Windows Media Çalışma Zamanı Modülü'nün kullanıldığı tüm sistemler bu güvenlik açığından etkilenir.
  
**Güncelleştirme ne yapıyor?**    
Güncelleştirme, ASF dosyalarının Windows Media Çalışma Zamanı Modülü tarafından işlenme biçimini değiştirerek bu güvenlik açığını ortadan kaldırır.
  
**Bu güvenlik bülteni yayımlandığında bu güvenlik açığı genel olarak duyurulmuş muydu?**    
Evet. Bu güvenlik açığı genel olarak duyurulmuştur. Ortak Güvenlik Açığı ve Etkilenme (Common Vulnerability and Exposure) numarası olarak [CVE-2009-0555](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2009-0555) atanmıştır.
  
**Bu güvenlik bülteni yayımlandığında, Microsoft bu güvenlik açığından yararlanıldığına dair herhangi bir rapor almış mıydı?**    
Evet. Microsoft bu güvenlik açığından yararlanmaya çalışan sınırlı sayıda saldırı olduğunu bilmektedir.

    `cacls “%WINDIR%\syswow64\msaud32.acm” /E /R everyone`
  
    Windows Vista ve Windows Server 2008 üzerinde:
  
    Komut isteminde aşağıdaki komutları yönetici olarak çalıştırın:

    `icacls %windir%\syswow64 /restore %TEMP%\msaud32_acm.TXT`

**Ses Sıkıştırma Yöneticisi nedir?**    
Ses Sıkıştırma Yöneticisi (ACM), saydam çalışma zamanı ses sıkıştırma, sıkıştırmayı açma ve Waveform ses verilerini işleme için sistem düzeyinde destek sağlar. ACM varolan sürücü arabirimi kancalarını kullanarak, dalga biçimi ses aygıtları için varsayılan eşleme algoritmasını geçersiz kılar. Böylece, aygıt ile açık çağrıların ACM tarafından önlenebilmesi sağlanır. Bir çağrı önlendikten sonra, ACM ses verilerini işlemek için çeşitli görevler gerçekleştirebilir; örneğin, sıralamaya harici bir sıkıştırıcı veya sıkıştırma açıcı ekleyebilir. ACM hakkında daha fazla bilgi için, [Ses Sıkıştırma Yöneticisi Hakkında](http://msdn.microsoft.com/en-us/library/dd742878(vs.85).aspx) adlı MSDN makalesine bakın.
  
**Bir saldırgan bu güvenlik açığından yararlanarak neler yapabilir?**    
Kullanıcı yönetimsel haklarla oturum açtıysa, bu güvenlik açığından başarıyla yararlanan bir saldırgan etkilenen sistemin denetimini tümüyle ele geçirebilir. Saldırgan, program yükleyebilir; verileri görüntüleyebilir, değiştirebilir veya silebilir; tüm kullanıcı haklarına sahip olan yeni hesaplar oluşturabilir. Hesapları, sistemde az sayıda kullanıcı hakkıyla yapılandırılmış olan kullanıcılar, yönetici haklarıyla çalışan kullanıcılardan daha az etkilenebilir.
  
**Bir saldırgan güvenlik açığından nasıl yararlanabilir?**    
Bu güvenlik açığından yararlanılabilmesi için, kullanıcının özel hazırlanmış bir medya dosyasını açması veya Web içeriği sağlayan bir Web sitesinden ya da başka bir uygulamadan özel hazırlanmış akış içeriği alması gerekir.
  
**Bu güvenlik açığından birincil derecede hangi sistemler etkileniyor?**    
İş istasyonları veya terminal sunucuları gibi, Windows Media Player'ın kullanıldığı tüm sistemler bu güvenlik açığından etkilenir.
  
**Güncelleştirme ne yapıyor?**    
Güncelleştirme, Windows Media Çalışma Zamanı Modülü'nün sıkıştırılmış ses dosyalarındaki işlevleri başlatma biçimini düzelterek bu güvenlik açığını giderir.
  
**Bu güvenlik bülteni yayımlandığında bu güvenlik açığı genel olarak duyurulmuş muydu?**    
Hayır. Microsoft, bu güvenlik açığı hakkında bilgilere sorumlu kişiler aracılığıyla ulaştı.
  
**Bu güvenlik bülteni yayımlandığında, Microsoft bu güvenlik açığından yararlanıldığına dair herhangi bir rapor almış mıydı?**    
Hayır. Microsoft, bu güvenlik açığının müşterilere saldırmak amacıyla açıkça kullanıldığına dair herhangi bir bilgi almadı. Ayrıca, bu güvenlik bülteni ilk yayımlandığında, açığın kanıtı olabilecek kod örneği yayımlandığını görmedi.
